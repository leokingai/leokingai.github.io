亚马逊中小卖家需要做Shopify吗？

发表于 2018-07-12 | |

Abstract：

亚马逊的竞争日益激烈，花了点时间研究了独立站shopify的运营，这里分享下我的心得体会。

什么是shopify?

Shopify是一个SaaS领域的购物车系统，适合跨境电商建立独立站，用户支付一定费用即可在线利用各种主题，模板，APP建立自己的网上商店。

在跨境电商中要建立独立站，我们在选择购物车系统时，特别是对于之前做平台（如Amazon，Wish，Aliexpress等）的人说更是困难。如果您刚好想建一个全新独立站，Shopify肯定是一个非常好的选择，因为它是创建独立站最简单最高效一种方式。（下面会解释为什么是最简单最高效）

Shopify的历史及取得的成绩：

Shopify 成立于2004年，当初开发这套系统是为了在线卖滑雪板，所有它的出生就带着非常纯正的电商血统。到2017年8月1日官方统计信息显示，Shopify在全球已经拥有50万个商家，为全球175个国家提供电商服务，从2012年到现在已经实现累计在线交易额400亿美金，并且每年保持平均74%的增长速度。

亚马逊卖家需要做shopify 吗？

Anker 的一些子品牌即是用shopify 建站：

shopify网站外观精美，功能可以通过应用商店的APP做成一个非常强大的独立站，去Youtube 花了点时间看了老外一大把教学视频，有做shopify 月销售几十万上百万刀的，实际操作了一把，发现并不容易，首先你得是个社交达人，有粉丝，当然你也可以通过Facebook广告，砸Instagram，youtube红人去引流，但是这些都并不容易，我最直观的感受是亚马逊卖家如果没有专门的SNS运营团队，还是专注产品，做好亚马逊比较现实，做好Shopify不像其它电商平台，可以自带流量，而只是一个建立独立网站的工具，需要通过商家的自运营来推广店铺。因Shopify整合Facebook和Twitter等信息平台，其店铺流量也主要来自于Facebook和Twitter的引流，或通过搜索引擎获取。

但是亚马逊卖家都需要一个官网来做品牌LOGO 的，listing 过一阵title 就被锁死了，什么都改不了，亚马逊会要你提供官网证明。

Shopify的5大优点：

1. 专业精美的Shopify模板：

Shopify官方提供超过60个精美，专业的模板，而且移动端全部采用自适应，完美兼容各种移动设备。

Shopify为您提供的精美而专业的模板主题，可以让您的独立站看起来更专业。 专业的模版，让你的网站得到更多的信任，以使访客留下来浏览更多的页面，获得更多的访问时长更多的订单。

Shopify为您提供了服装，珠宝，家具，艺术品等的主题，有免费也有付费的，最贵的180刀，是一次性付款永久使用。值得一提的是，Shopify邀请的都是最专业的程序员为Shopify创建主题, 这些主题由Shopify官方进行质量审核，因此它们能与Shopify完美兼容， 这么多主题中总有一款适合您的网站。

2. 功能齐全的APP市场：

这个也是我喜欢使用Shopify的最主要原因。

Shopify APP

Shopify的App跟我们的手机安装APP类似。就好比是你买一个最新款的iPhone 8，他只有基础的你打电话发短信功能，如果您要用手机看视频，玩游戏，刷个朋友圈那就得安装优酷土豆，王者荣耀，微信等APP应用。同样我们的我们的Shopify安装好后，只有基础的购物功能。如果你要批量管理产品，批量管理订单，高级的邮件营销等功能，那你就需要安装各种App。

这实际上是Shopify最大的优点之一。它不仅可以让您建立一个网上商店并处理付款，还可为您提供有用的工具，帮助您成功开展业务的其他重要方面，比如App可以从这些方面帮你：产品发布，产品库存管理，邮件营销，SNS营销，再营销，高级分析报告，SEO，订单管理等等方面，Shopify App Store里面有超过1000种不同应用供您选择，这些App有很大一部分可以助你提供效率以及自动化。比如我最爱用国产ERP店小秘,可以高效快速的管理订单，一键生成上百个E邮宝单号及快速打印单号。

您可以选择你需要的应用程序，这些应用程序可以完美集成到Shopify中，而且可以即插即用 ，安装后进行简单的设置即可使用，一般App都有详细简单的教程来教你怎么设置，这将节省您大量的时间和工作。付费方式灵活：这些App基本都有试用期1-30天不等，如果试用不满意，在试用期内卸载就不会扣费；收费方式基本上都是根据你的业务规模实行按月收费，以后你不想用了，觉得没有效果，直接一键卸载就可以了，以后就不会产生费用了。

3. Shopify为您提供全年365天，全天24小时在线支持：

您可以随时通过电话（国内很少用，毕竟打电话到国外费用比较贵），在线聊天或电子邮件（是我们国内最喜欢选择的沟通方式）与他们联系。比如你在设置collections时遇到问题，或者绑定域名遇到困难，你都可以点击在线聊天跟Shopify的官方技术支持及时解决你的问题，目前官方还不支持中文交流，请使用英文哦。

4. Shopify为您提供稳定安全的服务器，在全球部署免费高速的CDN：

速度：成功运行在线商店的因素之一，CWCS研究发现，如果网页加载时间超过三秒，57％的客户离开网站。点击这里测试一下的您的网站打开速度：

Shopify的首页打开速度测试结果

如果你手上有自己Shopify网站，你自己测试一下你的网站速度，不单单要测试首页，产品页面有时候速度显得更重要。

安全：因为我们的网站是电子商务网站涉及交易，所以我们的网站需要比一般新闻门户资讯类的网站要具有更高的安全性。

  1. Shopify全站采用SSL安全协议，保证传输数据的安全性（任何套餐都可以免费具有该功能，不需要额外付费）。

  2. Shopify 支付安全性已通过PCI DSS的认证，可以更好的保护交易信用卡持卡人的资料安全。

  3. Shopify还完美的整合了PayPal支付，不是所有的站点都可以接入PayPal支付，PayPal也对独立站的安全性有一定的要求，而使用Shopify你只需要1分钟就可以接入PayPal支付。

Shopify作为一个全面的电子商务解决方案，他们已经聘请了许多专业的工程师，让我们的网站不用担心被黑客攻击。此外Shopify是一家上市公司，所以你也有理由相信他们必然会在网站安全方面投入巨大资金。

5. Shopify 兼容多渠道销售：

只要你的Shopify店铺建立后，产品上传好后，配合相应的APP（部分渠道不需要额外的APP）及收款方式，你就可以在这些平台开始您的销售了，而不需要重复上传产品到这些平台上。这个功能非常强大，比如市面上说的“Facebook全球开店”，动不动开店费用就要上万元，如果你用了Shopify后，直接在后台开启Facebook销售渠道即可，这样所谓的“Facebook全球开店”就实现了，完全免费。

Shopify目前兼容的销售平台有：

官方介绍：https://www.shopify.com/channels

社交平台：Facebook，Pinterest，Facebook Messenger

专业销售平台：Amazon，Wish，Ebay（即将上线）

其他平台：Wanelo，BuzzFeed，Kik，Houzz等

Shopify的5大缺点：

1.Shopify的交易费用：

这个交易费我想是Shopify最致命的缺点，如果用Magento，Zencart来作为建站程序，完全没有交易费用，使用这2个购物车是100%免费的。Shopify除了每月的套餐费用外，Shopify还会向我们这些用户收取其计划的0.5％-2％的交易费用，除非您使用Shopify Payments为您的收款方式， 在这种情况下, 你所有交易费用都将被免除，但是我们国内很难做到采用Shopify Payment，开通Shopify Payment需要美国的身份证和该身份证对应的美国的银行账号进行验证。所以目前来说，我们暂时开通不了Shopify Payment，这就意味着我们每个月除了支付基础的月费外，如果还产生了交易，您还需要额外给Shopify将向您收取0.5-%2％的交易手续费。

举个3个例子：

假如我们使用29美金每月的Shopify基础方案，当月交易额3000美金。您要付给Shopify的费用为：29+30002%=29+60=*89USD

假如我们使用79美金每月的Shopify方案，当月交易额10000美金。您要付给Shopify的费用为：79+100001%=79+100=*179USD

假如我们使用179美金每月的高级Shopify方案，当月交易额50000美金。您要付给Shopify的费用为：179+500000.5%=179+250=*429USD

我们就说最后一个429USD的费用，显然这个看起来确实有点高了，但是我们计算一下我们的利润，我们假设我们的纯利润20%的话，我们每月也可以拿到10000美金的纯利润。这个收费也是非常透明的，每个月你的销售额一出来，你就可以计算出的你的交易费。还有你每月有50000美金的销售额，说明的网站访问量也增加了，相应的占用的资源也多了，比如宽带资源，而我们花429美金就可以保证我们网站的稳定性安全性，还有更重要的一点你都有10000美金的纯利润收入，你就不能花429美金买个安心呀，毕竟这个钱是花的值得的。

2. Shopify的App费用：

虽然Shopify为您提供超过1,400种应用程序来扩展您的在线商店的功能，但许多应用程序不是免费的，并且每月向您收取费用以供使用。显然这可能会让你的成本继续增加。例如，将Cross Sell 和Social Login集成到Shopify商店你每月需要额外增加9.9+1.49=11.39USD。

举例说明一下这个APP费用，我们如何才能心甘情愿把这些费用掏给Shopify的开发者：

Hurrify - Countdown Timer 这个APP如何帮我节省大量时间，增加效率？这个是一个产品页面促销倒计时APP，可以让客户产生紧迫感，如果不下单促销可能马上就要结束了。在安装这个APP之前，我用ELLE模板自带的倒计时功能，它有个弊端就是需要你在每次促销时间结束后，重新手动在设置，这样就费用了非常多的时间。比如我明天有50个产品要参加这个倒计时促销活动，就要手工点击50次来修改倒计时，非常浪费时间。但是安装Hurrify后，只需要一次设置，永久有效，效率当然提高了N倍，我当然愿意，而且非常高兴能每月用6.99美金来换我的时间。

一部分的APP会使您的工作更轻松 ，获得更高的效率；一部分的APP会让你的网站更专业，获得更多的销售，拿到更多的流量，获得更多邮件订阅，而且应用程序与Shopify完美集成，你完全不必要担心BUG，直接联系开发者即可，他们回复的效率也很高。因此您可以专注于业务的其他重要方面。如果还担心APP费用，请记住APP是完全可选的。

3. 网站程序采用Liquid语言，专业的开发程序员少：

目前在国内很难找到非常专业的Liquid程序员，本身这个程序的教程基本是英文的。Liquid语言比较小众，不像PHP，国内已经有很多高手，也有很多这样的成熟的中文课程。不过您要对网站进行修改，或者您要开发特定的功能，您可要在Shopify官方寻找专家，这些涵盖了：网站设置，模板设计，专业开发，营销，图片设计等专家，在这里还可要找到部分中国的开发者。

Shopify 官方专家服务：https://experts.shopify.com/

4. Shopify的备份转移：

我目前还没有想到要迁移Shopify，但是这并不代表我以后会一直用Shopify，万一有更好的购物车程序呢？如果真的要迁移网站，现在能做的方法只能后导出CSV格式的产品，然后再开一个Shopify，重新导入产品，但是对于博客内容，联系我们，退换货政策等页面呢，这些页面也就很难恢复了。我目前还没有想到更好的方法。

5.批发功能基本没有涉及到，多语言的支持不是很好：

我现在有做B2B小额批发，所以会用到Shopify来做这种小B2B，但它没有这个批发功能，比如我要对价格进行阶梯设置，Shopify需要借助APP，而在Mangeto中可以很容易的实现，因为Magento本身自带这个功能。

网站多语言的问题，Shopify目前基本只能靠App来解决，如langify。

结论：

综上所述：您认为优点是否胜过了缺点？ 毫无疑问，我个人认为上述提到的几个缺点并不是真正无法接受，这些缺点只会影响很小一部分用户。 在遇到Shopify前，我从未想过推出一个在线商店如此简单， 这对大多数不懂代码，想用独立站来试试手的跨境电商中小从业者来说是个好消息， 只需要每个月29美元，都可以用Shopify建立一个专业的电子商务网站。赶紧来试试吧，14天免费试用，不需要绑定信用卡。
